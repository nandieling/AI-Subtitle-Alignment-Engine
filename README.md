# 🎬 AI 字幕对齐引擎 Pro (AI Subtitle Alignment Engine)

一款基于强大的自然语言处理 (NLP) 模型构建的本地化 Mac 桌面软件，专为字幕组、影视译者和视频创作者设计。彻底告别手动打轴的痛苦，让 AI 跨越语言壁垒，为你实现毫秒级的中外文字幕智能对齐。

![](https://img2.pixhost.to/images/6410/704282331_1.png)

## ✨ 核心特性 (Features)

* 🧠 **纯本地化 AI 运算:** 核心搭载 Hugging Face 的 `paraphrase-multilingual-MiniLM-L12-v2` 多语言模型。首次下载后完全离线运行，无需 API 密钥，拒绝网络波动，保护字幕隐私。
* 🎯 **DTW 动态时间规整算法:** 不依赖简单的按行匹配，而是将句子转换为高维数学向量，通过计算“余弦相似度”找到多语言之间最精准的语义对应关系。
* 🎨 **ASS 特效无损保留 (独家):** 行业领先的“剥离-还原”双规引擎。自动屏蔽 `{\an8}`、颜色代码等 ASS 特效标签进行纯净的 AI 语义理解，对齐完成后原封不动地将特效贴回全新时间轴，排版不乱！
* ✂️ **智能断句与异常修剪:** 内置最大时长断点（默认 6 秒），完美解决长镜头、纯音乐过场导致的“幽灵长字幕”残留问题，让字幕干净利落。
* 🖥️ **原生极简 GUI:** 基于 PyQt5 打造的优雅桌面级 UI。支持文件拖拽、一键点击，告别繁琐的命令行黑框。支持 `.srt` 与 `.ass` 格式自由混搭对齐。

## 🚀 快速使用 (Usage)

1. **准备文件:** 准备一份时间轴准确的**外文基准字幕**，以及一份时间轴错乱的**中文翻译字幕**（支持跨格式，例如 SRT 对齐 ASS）。
2. **拖入软件:** 打开应用，将两个文件分别拖入对应的虚线框内（或点击虚线框手动选择文件）。
3. **一键对齐:** 点击底部的“开始对齐”按钮，选择保存路径。
4. **见证奇迹:** 后台多线程将自动进行向量转换与动态规划，仅需几秒至几十秒，即可收获完美卡点的全新中文字幕！

## 🛠️ 开发者指南 (For Developers)

如果你希望从源码运行或二次开发本项目，请按以下步骤配置环境：

### 1. 安装依赖
请确保你的机器上已安装 Python 3.8+，然后在终端执行：
```bash
pip3 install -r requirements.txt
```
### 2. 从源码运行

```Bash
python3 app.py
```

### 3. 打包为 Mac 原生应用 (.app)

本项目使用 PyInstaller 进行独立打包。在项目根目录下准备好你的 icon.icns 图标文件，然后执行以下命令：

```Bash
pyinstaller --name "AI字幕对齐引擎" --windowed --noconsole --icon="1.icns" --clean app.py
```

打包完成后，即可在 dist 目录下找到可双击运行、可随意分发的独立 Mac 软件。

📝 算法原理解析
文本清洗: 提取 SRT/ASS 中的纯净文本内容，保存原有的元数据（Metadata）和样式表。

向量化 (Embedding): 利用 Sentence-Transformers 模型，将中外文字幕分别映射到相同的多维语义空间中。

距离矩阵: 计算每一句中文与每一句外文之间的语义距离矩阵。

动态规划 (DTW): 在距离矩阵中寻找一条从左上角到右下角的“成本最低”路径，解决“一句外文对多句中文”或“多句外文对一句中文”的复杂情况。

重塑与裁切: 根据最优路径重新分配时间戳，并应用防超时修剪机制，最后封装回原始文件格式。
